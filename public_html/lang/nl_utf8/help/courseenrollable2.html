Deze instelling bepaalt of een leerling zichzelf kan aanmelden bij een cursus via de standaard interactieve aanmeldingsplugin (bv Interne aanmelding) of die leerling de melding krijgt: "Je kunt je nu niet aanmelden bij deze cursus."

Deze instelling heeft geen effect op de niet-interactieve aanmeldigsplugins (vb externe databank aanmelding). Bovendien kunnen gebruikers altijd de rol van leerling gegeven worden via de Rollen toewijzen link in het beheerblok van een cursus.
