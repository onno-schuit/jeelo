Kies filter

Met deze filter kun je informatie zoeken, gebaseerd op een rolmenu. De filter heeft volgende opties:

  • alle waarden - deze optie schakeld de filter uit (want alle informatie wordt door deze filter aanvaard)
  • is gelijk aan - deze optie toont enkel informatie die gelijk is aan de waarde, geselecteerd uit de lijst
  • is niet gelijk aan - deze optie toont enkel informatie die verschilt van de waarde, geselecteerd uit de lijst
