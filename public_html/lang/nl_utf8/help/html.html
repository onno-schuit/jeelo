Hulp bij het schrijven van HTML

Je kunt in Moodle alle HTML-tags gebruiken die je nodig hebt om het effect te bekomen dat je wenst.

Merk op dat scripting, zoals Javascript of VB-script niet toegelaten is en automatisch verwijderd zal worden.

Je code zal op de pagina getoond worden binnen de cel van een tabel, dus

  • het is niet nodig om <HEAD> or <BODY> tags te gebruiken
  • >wees voorzichtig met het verkeerd gebruik van </table> tags die de weergave van de pagina kunnen verknoeien.

Emoticons zullen in hun grafisch equivalent geconverteerd worden en URL's zullen in links omgezet worden.
