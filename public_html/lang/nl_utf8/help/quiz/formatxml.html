Categorie schakelen

Je kunt een speciale vraag in je XML-bestand stoppen, die de informatie bevat over de categorie waarin de volgende vraag moet geïmporteerd worden. Als de categorie of categorieën niet bestaan, dan worden ze aangemaakt. Merk op dat de uit bestand-optie moet aangevinkt zijn, anders wordt deze code genegeerd. Je kunt in het XML-bestand zo dikwijls van categorie wisselen als je maar wil.

Een voorbeeld van deze opmaak is:

<question type="category">
    <category>
        tom/dick/harry
    </category>
</question>
