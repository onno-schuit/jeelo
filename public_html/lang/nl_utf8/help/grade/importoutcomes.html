Competenties importeren

Je kunt competenties importeren via .csv-bestanden. De opmaak van de bestanden moet als volgt:

Veldnaam Beschrijving Vereist Opmaak
outcome_name De volledige naam van de competentie JaString
outcome_shortnameDe korte naam voor de competentie Ja String
outcome_descriptionDe beschrijving van de competentie Nee String
scale_nameDe naam van de gebruikte schaal Ja String
scale_itemsEen kommagescheiden lijst met schaalitems Ja String
scale_descriptionDe beschrijving van de schaal Nee String

Hier is een voorbeeld:

outcome_name;outcome_shortname;outcome_description;scale_name;scale_items;scale_description
Deelname;deelname;;Participation scale;"Weinig of geen deelname,genoeg deelname,Enthousiaste deelname";
