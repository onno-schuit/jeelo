Gebruikerssleutel

Cijfers publiceren is een manier om cijfers te exporteren via een URL zonder aangemeld te zijn in Moodle. Een gebruikerssleutel is een random tekenreeks die aan de URL wordt toegevoegd voor beveiligingsredenen.

Om cijfers te publiceren kies je "Maak een nieuwe gebruikerssleutel". Als het formulier wordt ingestuurd, wordt een nieuwe gebruikerssleutel gemaakt. Als alternatief kan een eerder aangemaakte gebruikerssleutel gekozen worden.
